* Introduction

- This is the story board for the UI analytics of the assigned project.

* Scope of Project

  To do the UI analysis of all the expermients of vlabs - IIIT Hyderabad.
  
  1) To analyse traffic on the vlabs website.

  2) To collect data and present in a form so that it is
     easily consumable.
  3) To save data in form of reports.

* Motivation

  1) Being an experiment related to education sector, it
     becomes really important to know the reach of project
     and how useful it is for the consumers. To know which UI
     analysis is necessary.

  2) Moreover, records and statistics from UI analysis are
     very helpful in improving the experiment.
