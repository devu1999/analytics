* Final Verdict

- given the current state of Matomo it simply does not offer any
  exceptional, unique features that Google Analytics cannot provide.

- Considering a website without such an enormous traffic ,Google
  Analytics will do a great job satisfying your analytics
  requirements. It is easy to set up, and you can be up and running in
  literally 15 minutes.

- Matomo is a good second alternative, but if you are someone who is
  going to use the basic set of data and just need to get on with it,
  then Google analytics should be the go-to software.

*Google Analytics definitely has an upper edge over Matomo*
